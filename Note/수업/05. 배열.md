# 배열

- 배열은 연속된 메모리 공간을 확보하며, 이름은 첫 요소를 가리키는 포인터처럼 작동.
    
    ```cpp
    int example[5];
    
    int* ptr = example;
    *(ptr + 2) = 20;                   // example[2] = 20; 와 동일
    *(int*)((char*)ptr + 8) = 20;      // 바이트 단위로 접근 (4바이트 * 2 = 8)
    ```
    
- 배열의 크기를 벗어난 접근은 미정의 동작이므로 항상 경계(bound) 확인이 필요하다.
- 동적 배열은 `new[]`로 할당하고, `delete[]`해야 메모리 누수를 방지할 수 있다.
